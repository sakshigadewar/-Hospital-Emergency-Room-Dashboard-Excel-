📊 Overview

The Hospital Emergency Room Dashboard is an interactive Excel-based reporting tool designed to analyze patient flow, wait times, and hospital performance metrics. It helps stakeholders quickly understand key insights and improve decision-making in emergency healthcare services.

---
🎯 Objective

Efficient management of emergency room operations is critical for improving patient care and hospital performance. This project aims to:
- Identify delays in patient handling and reduce waiting time
- Analyze patient admission trends and resource utilization
- Evaluate patient satisfaction to improve service quality
- Determine the most frequently referred departments
- Understand patient demographics for better planning
- Identify operational inefficiencies in emergency room workflow

---
🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers (Month & Year filter)
- Conditional Formatting
- Data Cleaning Techniques

  📁 Dataset

The dataset used in this project includes:
- Patient ID
- Age Group
- Gender
- Admission Status
- Wait Time
- Department Referral
- Visit Date (Month & Year)
- Satisfaction Score

📌 Key Features

- Interactive Dashboard with dynamic filters for month and year
- KPI Tracking including total patients, average wait time, and satisfaction score
- Admission Analysis showing admitted vs non-admitted patients
- Wait Time Insights to identify delays and improve efficiency
- Patient Demographics Analysis (age group & gender distribution)
- Department Referral Analysis to track patient distribution across departments
- Attendance Status Monitoring (on-time vs delayed patients)
- User-Friendly Visualization using charts, slicers, and conditional formatting

  🚀 How to Use

- Download or clone this repository
- Open the Excel file (ER_Dashboard.xlsx) in Microsoft Excel
- Navigate to the dashboard sheet
- Use slicers (Month & Year) to filter data dynamically
- Analyze KPIs and charts to gain insights into patient flow, wait times, and performance metrics
